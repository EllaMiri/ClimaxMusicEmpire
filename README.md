# Examensarbete Climax Music Empire

## Getting Started

Start by installing all the dependencies with:

```
npm i
# or
yarn i
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
